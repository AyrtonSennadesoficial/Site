Passos para configurar o git

  1. Verificar as credenciais do Windows
  2. Configurar o nome e email do git
  3. Listar as configurar

Comando:

Verificar:
git config --list
Configuração:
git config --global user.name "seu nome"
git config --global usre.email "seu email"

Passos para o git monitorar uma pasta e salvar uma versão do projeto

  • criar umna pasta para o projeto no computador
  • git init
  • git add nome do arquivo
  • git commit -m"mensagem"

Passos para enviar o projeto salvo no computador para o Github

  • git branch -M Main
  • git remote add origin
